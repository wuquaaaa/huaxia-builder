# 华夏建造者（Huaxia Builder）设计文档

> 一款以**古代中国**为背景的放置/增量经营游戏（Idle / Incremental Game）。玩家从一处荒村、寥寥几户人家起步，逐步开垦农田、伐木采矿、兴办书院、敬天祭祖，营建一个由村落走向盛世王朝的文明：采集粮食、营造屋舍、研习学问、调度百姓、应对四季荒歉，最终通商西域、远洋开拓，并在"改朝换代"中代代相传、国运愈盛。
>
> 玩法框架源自经典放置游戏《Kittens Game》（社区译名"猫国建造者"），本作将其完整中式化：以**粮食**替代猫薄荷、以**农田**替代猫薄荷田、以**百姓（人）**替代猫，并将贸易、宗教、转生、终局等系统全部重铸为古代华夏设定。

---

## 1. 设计概述

### 1.1 核心定位
- **类型**：放置 / 增量 / 文明经营（Idle / Incremental / Civilization Management）
- **题材**：古代中国（先秦至明清的泛历史想象，不绑定具体朝代）
- **平台**：网页（HTML5 + TypeScript），可移植到移动端
- **视角**：以数值面板与古朴 UI 为主，无即时战斗，强调"看着村落兴旺、数字增长"的满足感
- **节奏**：前期手动采集，中期屋舍/作坊自动产出，后期靠学问、通商与"改朝换代"循环精进

### 1.2 核心循环（Core Loop）
```
采集粮食 → 营造屋舍/农田 → 研习学问 → 增加人口 → 调派工役 →
提升产能 → 解锁更高阶资源/建筑 → 应对瓶颈（荒年/仓廪上限/香火）→
改朝换代（重置）获得永久国运加成 → 更快地循环
```

### 1.3 设计目标
1. **正反馈密集**：每一次开垦、每一次升迁都有可见数值回报。
2. **决策有深度**：资源相互转化，仓廪有上限，工役分配需要权衡。
3. **长线留存**：通过"改朝换代（转生）"系统提供"重开更强、国运日盛"的长期目标。
4. **离线友好**：关闭游戏后产出仍按比例累积。

## 2. 资源系统（Resources）

资源是游戏的核心。分为基础资源、加工资源、稀有资源与特殊资源四类。每种资源都有**产量（per tick）**与**仓廪上限（cap）**两个关键属性。

### 2.1 基础资源
| 资源 | 来源 | 说明 |
|------|------|------|
| 粮食（Grain） | 采集 / 农田 | 最基础的食物，养活百姓，冬季消耗加剧 |
| 木材（Wood） | 樵夫 / 伐木 | 营造核心材料 |
| 矿石（Minerals） | 矿工 | 营造与冶炼材料 |
| 毛皮（Furs） | 狩猎副产 | 贸易与衣物材料 |

### 2.2 加工资源
| 资源 | 来源 | 用途 |
|------|------|------|
| 铁（Iron） | 冶炼坊消耗矿石 | 农具、兵器、进阶营造 |
| 煤（Coal） | 煤窑 / 深矿 | 冶炼与燃料 |
| 钢（Steel） | 铁 + 煤 | 高级营造、远洋 |
| 石料（Stone） | 采石加工 | 大型工程地基（城墙、宫室） |
| 丝绸（Silk） | 织造坊 / 蚕桑 | 高价值贸易品 |

### 2.3 稀有 / 特殊资源
| 资源 | 说明 |
|------|------|
| 黄金（Gold） | 贸易、敬神、高级转化的硬通货 |
| 玉石（Jade） | 西域通商所得，礼器与珍宝 |
| 香料（Spice） | 丝绸之路贸易所得，提升民心 |
| 文化（Culture） | 戏台 / 书院产出，影响学问与外交 |
| 学问（Knowledge） | 书生产出，用于研习科技（学术树） |
| 香火 / 功德（Faith） | 道观 / 道士产出，信仰系统核心 |
| 玄铁（Mythril） | 远洋秘境所得，后期顶级材料 |

### 2.4 仓廪上限机制
- 每种资源都有 cap，超出部分被浪费。
- 提升 cap 是中期核心目标：库房（Warehouse）、粮仓（Granary）、码头（Harbour）等专门用于扩容。
- 设计意图：迫使玩家在"扩建仓廪"与"消耗产出"之间做权衡，避免无脑囤积。

## 3. 建筑系统（Buildings）

建筑是产能的来源，绝大多数建筑成本随数量呈**指数增长**（典型系数 1.12～1.15），形成天然的成长曲线。

### 3.1 生产类
- **农田（Farmland）**：产粮食，受季节系数影响（春+，冬-）。
- **茅屋 / 民居（House）**：增加人口上限（每座 +2 人），百姓需消耗粮食。
- **民居升级 → 瓦房 / 宅院**：提供更高人口上限与少量民心。

### 3.2 加工类
- **伐木场（Lumber Mill）**：提升木材产量。
- **冶炼坊 / 铁匠铺（Smelter）**：消耗矿石与木材产出铁（消耗型建筑，可开关）。
- **织造坊 / 水力工坊**：后期提升整体产能或转化稀有资源。

### 3.3 存储类
- **库房 / 粮仓 / 码头**：提升各类资源 cap。
- 设计上需要玩家持续投入，否则产能会被 cap 卡死。

### 3.4 学问与文化类
- **书院（Academy）**：提升学问 cap 与产量。
- **太学（Imperial College）**：进一步提升治学效率。
- **戏台 / 茶楼（Theatre）**：产出文化、安抚民心。
- **道观 / 庙宇（Temple）**：产出香火与文化，信仰系统入口。

### 3.5 高级 / 后期建筑
- **水力工坊、天工院、观星台**：提升产能与全局加成。
- **市舶司、宝船坞、远洋港**：解锁远洋开拓玩法。

> **设计要点**：建筑分"被动产出"与"开关消耗型"两类。消耗型建筑（如冶炼坊）让玩家主动管理燃料与原料平衡，增加策略性。

## 4. 人口与工役系统（Villagers & Jobs）

### 4.1 人口增长
- 百姓（人）随时间在"人口上限"内自然增长，前提是粮食储备为正。
- 食物不足时会触发**饥荒**，百姓会饿死流亡——这是核心的负反馈约束。

### 4.2 工役分配
玩家将每位百姓分派到一种差事，决定其产出：

| 工役 | 产出 | 解锁条件 |
|------|------|----------|
| 农夫（Farmer） | 粮食 | 初始 |
| 樵夫（Woodcutter） | 木材 | 初始 |
| 书生（Scholar） | 学问 | 研习科技 |
| 猎户（Hunter） | 毛皮 / 玉石 | 解锁狩猎 |
| 矿工（Miner） | 矿石 | 解锁采矿 |
| 工匠（Craftsman） | 加工 / 稀有 | 进阶 |
| 道士（Priest） | 香火 / 文化 | 信仰 |

### 4.3 技艺成长
- 每位百姓从事某差事越久，**技艺（skill）**越高，产量加成提升。
- 设计意图：让玩家不愿轻易调换工役，形成"养成"黏性；同时"乡贤 / 里正"系统可指定一人为首领获得全局加成。

### 4.4 民心（Happiness）
- 受人口数量（多则降低）、奢侈品消费（毛皮、玉石、香料、丝绸等提升）、文化建筑影响。
- 民心直接乘算到所有产出，是后期重要的优化维度。

---

## 5. 学问与文化系统（Knowledge & Culture）

### 5.1 学术树（Tech Tree）
- 用**学问**研习，呈树状依赖。
- 关键节点示例：农耕 → 畜牧 → 采矿 → 冶金 → 营造 → 算学 → 天文 → 医药 → 水利 → 火药 → 航海。
- 每研习一项学问，会开放新建筑、新工役、新资源或全局加成。

### 5.2 百工 / 作坊（Workshop / Crafting）
- 作坊允许将基础资源**加工合成**为进阶资源（如木材→石料、铁+煤→钢、蚕丝→丝绸）。
- 作坊升级提升合成效率、降低损耗、提升 cap。

### 5.3 文化与外交
- 文化值影响外交选项与部分学问解锁。
- 后期"治国之策（Government）"可选择仁政、法治、休养生息等方略，提供不同的全局倾向加成。

---

## 6. 季节与历法（Seasons & Calendar）

### 6.1 四季循环
游戏内时间以"天 → 季 → 年"推进，对应农历四时：

| 季节 | 影响 |
|------|------|
| 春 | 粮食产量 +50%，春耕黄金期 |
| 夏 | 正常产量 |
| 秋 | 正常，秋收备荒 |
| 冬 | 粮食产量大幅下降（-75%），饥荒压力陡增 |

### 6.2 天时事件
- 随机出现"风调雨顺""大旱""寒冬"等天时，进一步上下浮动产量。
- 设计意图：制造资源储备规划压力，玩家需在丰年囤粮以渡灾荒。

### 6.3 历法升级
- 研习"历法"后可预知节气与换季，进而规划农田与仓储节奏。

---

## 7. 贸易与外交（Trade & Diplomacy）

### 7.1 邦国 / 商队（Trade Partners）
通过开拓与遣使，解锁可通商的邦国与商队，各有特产与偏好：

| 邦国 / 商队 | 特产 | 偏好 |
|-------------|------|------|
| 西域商队 | 玉石 / 良马 / 香料 | 丝绸 / 黄金 |
| 草原部落 | 毛皮 / 马匹 | 粮食 / 铁器 |
| 海外番邦 | 香料 / 珍宝 | 丝绸 / 瓷器 |
| 南方诸国 | 茶 / 木材 | 铁器 / 食盐 |

### 7.2 贸易机制
- 每次通商消耗黄金 + 特定资源，按概率换回对方特产。
- 邦交（Standing）影响成功率与产出，可通过遣使纳贡提升。
- 设计意图：通商是获取**玉石、香料、良马**等本地难产资源的主要途径，构成中期资源闭环的关键。

---

## 8. 信仰系统（Religion）

### 8.1 香火积累
- 道士与道观产出**香火 / 功德**。
- 香火可"敬献"转化为**德行（Epiphany）**，进而升华为永久加成。

### 8.2 修行与教化（Transcendence）
- 高级信仰升级提供全局产能、仓廪与外交的乘算加成。
- 是中后期"质变"的重要来源。

### 8.3 天道 / 民间信仰
- 提供两条不同的信仰升级路线（如"敬天"重生产、"祭祖"重仓廪），鼓励玩法多样化。

---

## 9. 改朝换代与元进度（Reset & Meta-progression）

放置游戏的灵魂在于"重开更强"。本作以**王朝更替**承载这一机制，主题契合。

### 9.1 改朝换代（Reset）
- 玩家可主动"开创新朝"，将累计功业转化为**国运（Paragon）**与**声望 / 功德（Karma）**。
- 国运提供永久的产量与仓廪百分比加成，改朝换代后保留，谓之"气运绵延、代代相传"。

### 9.2 设计意图
- 第一个王朝可能需要数十小时，更替后凭借国运加成可在数小时内重达原进度并突破更远。
- 形成"螺旋上升"的长期目标，是留存的核心。

---

## 10. 远洋开拓（Endgame · 后期内容）

- 研习航海与营造宝船后开放远洋面板：建造市舶司、宝船坞，遣船队远航海外秘境。
- 远洋建筑产出顶级资源（玄铁等），并解锁全新学问分支（如水利大工、天文历算之极）。
- 作为终局内容，为硬核玩家提供长达数百小时的追求目标。
- （可选风味分支：另设"修行 / 登仙"路线，以香火与玄铁追求超凡，供后续扩展。）

## 11. UI / UX 设计

### 11.1 界面布局（古朴风格：宣纸底、木框、印章字体点缀）
```
┌────────────────────────────────────────────┐
│  资源栏（顶部常驻）：实时显示各资源/产量/上限   │
├──────────────┬─────────────────────────────┤
│  标签导航      │   主面板（按标签切换）         │
│  · 村落        │   · 建筑列表（可营造）         │
│  · 民生        │   · 工役分派                  │
│  · 学问        │   · 学术树                    │
│  · 通商        │   · 贸易 / 外交               │
│  · 信仰        │   · 信仰升级                  │
│  · 远洋        │                              │
├──────────────┴─────────────────────────────┤
│  纪事区：天时、换季、饥荒、人口等事件提示       │
└────────────────────────────────────────────┘
```

### 11.2 交互原则
- **悬停提示**：所有建筑/学问显示成本、产出、收益预测。
- **批量操作**：支持一次营造 ×1 / ×10 / ×最大。
- **颜色编码**：资源接近上限变黄、净产为负变红。
- **美术基调**：水墨/宣纸质感、毛笔字标题、朱印点缀，营造古意。

## 12. 技术实现建议

### 12.1 架构
- **状态层**：单一全局 game state 对象（资源、建筑、学问、人口等），易于存档序列化。
- **tick 循环**：固定时间步（如每秒 5 tick），每 tick 计算产出、消耗、增长。
- **离线结算**：记录退出时间戳，重进时按差值批量结算（设上限防滥用）。

### 12.2 数据驱动
- 建筑、学问、资源全部用配置表（JSON / TS）定义，逻辑与数据分离，方便平衡调整与扩展。

### 12.3 存档
- localStorage 自动存档 + 手动导出/导入（Base64 字符串），无需服务器。

### 12.4 数值平衡
- 建筑成本指数系数控制成长曲线（建议 1.12～1.15）。
- 用电子表格预先建模前 50 次营造的成本与产出，确保不会过快或过慢。

## 13. 开发里程碑（建议）

| 阶段 | 内容 |
|------|------|
| MVP | 粮食采集 + 农田 + 民居 + 人口 + tick 循环 + 存档 |
| Alpha | 完整资源链（木/矿/铁）+ 工役 + 学术树 + 季节 |
| Beta | 通商 + 信仰 + 民心 + 离线结算 + UI 打磨 |
| 1.0 | 改朝换代/国运系统 + 数值平衡 + 成就 |
| 1.x | 远洋开拓 + 终局内容 + 移动端适配 |

## 14. 附录：核心公式参考

- **建筑成本**：`cost = baseCost × ratio^(已建数量)`，ratio ≈ 1.12
- **资源产量**：`产出 = 基础产量 × 数量 × (1 + 技艺) × 季节系数 × 民心 × 全局加成`
- **人口增长**：`Δpop = growthRate × (当前人口) × (1 - 当前/上限)`，食物为负时转为饿亡
- **离线收益**：`offline = min(实际离线时长, 上限) × 每秒净产`
